### variables

assignment:
``` 
var a;
var b = "string";
var c = 1 + 2 +3;
var t = true;
var h =function(){};
```

``` var ``` - common variable that can be reassigned but only accessed within a function.

``` const ``` - var which cannot be reassigned. 

``` let ``` - var that can be reassigned but declared only once in the code

```
var age = 36
const pi = 3.14
let it = "be"
```
